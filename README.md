# وصي (Wasi)

  **Islamic Digital Will & Heritage Management Platform**  
  منصة إدارة الوصية الإسلامية الرقمية

  > وقف بنية عاصم عبدالرحمن محمد عمر — مهداة إلى عائلته وروح عمه عاطف حسن

  ---

  ## Stack

  - **Mobile App** — Expo React Native (Arabic RTL) + Firebase Auth + Firestore
  - **Web App** — React + Vite + Tailwind CSS + shadcn/ui (Arabic RTL)
  - **API Server** — Express 5 + Drizzle ORM + PostgreSQL

  ## Features

  - Write and store encrypted wills (خزنة الوصية)
  - Manage trusted guardians/executors (الأوصياء)
  - Track debts and religious duties like zakat (سجل الديون والواجبات)
  - Document digital legacy (الإرث الرقمي)
  - Dead man's switch system (نظام الضامن)

  ## Mobile App — Firebase Backend

  - **Firebase project**: `wasi-e1f16`
  - **Auth**: Email/Password via Firebase Auth
  - **Firestore**: Hierarchical `users/{uid}/subcollections` structure
